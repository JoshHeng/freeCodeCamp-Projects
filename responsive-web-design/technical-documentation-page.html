A Quick Summary of Python 3.7

Introduction

Python is a high-level, interpreted programming language which is rising in popularity, especially amongst schools. The syntax is very easy to read, and follows most coding conventions. The example code below accepts a number, and returns the number multipled by 5.

number = int(input("Please enter the number: ")) result = number * 5 print("The result is " + result)
Variables

There are several types of variables in Python:

  • Strings - text
  • Integers - whole numbers
  • Floats - numbers with decimal points

Converting between different variables is called casting. You need to do this in Python to convert user-inputted numbers to integers and floats. The syntax for casting to integers is int(string) and the syntax for casting to floats is float(string) a

integer = int(input("Integer: ")) float = float(input("Float: "))
Input and Output

The input() function is used to get user input as a string.

For example, name = input("Please enter your name") will save the user's name to the name variable.

To output something to the console, use print(string)

For example, print("Hello World") will display Hello World in the console.

Mathematical Operators

Python features the four common mathematical operators:

  • Addition: +
  • Subtraction: -
  • Multiplication: *
  • Division: /

The code below shows examples for each operator.

  • result = 5 + 5 10
  • result = 5 - 5 0
  • result = 5 * 5 25
  • result = 5 / 5 1
IF Statements

IF Statements control the flow of code by executing code only if a condition or several conditions are met.

The below code will accept a number from the user, and output if it is higher or lower than 10:

number = int(input("Please enter a number: ")) if (number < 10): print("This number is less than 10") elif (number > 10): print("This number is more than 10") else: print("This number is equal to 10")
